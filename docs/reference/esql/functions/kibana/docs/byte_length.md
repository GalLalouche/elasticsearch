### BYTE_LENGTH
Returns the byte length of a string.

```
FROM employees
| KEEP first_name, last_name
| EVAL fn_byte_length = BYTE_LENGTH(first_name)
```
Note: All strings are in UTF-8, so a single character can use multiple bytes.
